# ReactNative Movies
这是一个基于豆瓣Api使用 [ReactNative](https://github.com/facebook/react-native) 开发的App。
（目前只适配 iOS）
- react: 16.3.1
- react-native: 0.55.4
- 运行环境：iOS 8.0 或以上（暂未适配 Android）
